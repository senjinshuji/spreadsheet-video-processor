Video Processor Help

Quick Start

  1. Set up your spreadsheet with media URLs and durations
  2. Configure settings (Video Processor → Settings)
  3. Select rows to process
  4. Click Video Processor → Process Selected Rows
  5. Wait for processing to complete
  6. Download your processed videos from the download links

New Spreadsheet Format

The new format uses a flexible material-based structure:

Column Required Description Example
実行対象 Yes Execution target marker ○
実行結果 Auto Processing status and final video URL https://api.../download
素材① (URL) Yes First material URL video1.mp4
使用開始 Yes Start time for material 0:05 or 5
使用秒数 Yes Duration to use 0:10 or 10
素材②... No Additional materials (repeating pattern) image1.jpg, 0:00, 0:03

Features

Multiple Media Files

You can process multiple media files in a single row by separating URLs and durations with commas:

Example:
URLs: video1.mp4,image1.jpg,video2.mp4
Durations: 5,3,10

Range Selection

Select multiple rows to process them as separate videos:

  • Click and drag to select multiple rows
  • Use Ctrl+Click (Cmd+Click on Mac) to select non-consecutive rows
  • Each row will be processed as a separate video

Supported Formats

Video: MP4, AVI, MOV, MKV, FLV, WMV, WebM

Image: JPG/JPEG, PNG, GIF, BMP, TIFF

Settings

Column Mappings

Configure which columns contain your data. Use column numbers (A=1, B=2, etc.)

Output Settings

  • Format: Output video format (MP4 recommended)
  • FPS: Frames per second (30 is standard)
  • Resolution: Output resolution (e.g., 1920x1080)
  • Quality: Balance between file size and quality

Tips

  • Use consistent video resolutions for best results
  • Images will be displayed for the specified duration
  • Processing time depends on video length and quality settings
  • You can close the processing dialog - work continues in background
  • Check the status column for real-time updates

Troubleshooting

Common Issues

  • API Connection Failed: Check your API URL in Settings
  • Processing Failed: Verify media URLs are accessible
  • No Download Link: Processing may still be in progress
